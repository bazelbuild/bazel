DottedVersion

A value representing a version with multiple components, seperated by periods, such as 1.2.3.4.

Methods

  • compare_to

compare_to

int DottedVersion.compare_to(DottedVersion)

Compares based on most signifigant (first) not-matching version component. So, for example, 1.2.3 > 1.2.4
