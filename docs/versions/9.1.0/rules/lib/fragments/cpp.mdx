## custom_malloc

```
Label cpp.custom_malloc
```

Returns label pointed to by [`--custom_malloc`](/versions/9.1.0/docs/user-manual#flag--custom_malloc) option. Can be accessed with [`configuration_field`](../globals/bzl#configuration_field):

```
attr.label(
    default = configuration_field(
        fragment = "cpp",
        name = "custom_malloc"
    )
)
```

May return `None`.
