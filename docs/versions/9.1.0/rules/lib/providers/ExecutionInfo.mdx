## ExecutionInfo

```
ExecutionInfo ExecutionInfo(requirements={}, exec_group='test')
```

Creates an instance.

### Parameters

| Parameter | Description |
| --- | --- |
| `requirements` | [dict](/versions/9.1.0/rules/lib/core/dict); default is `&lcub;&rcub;`  A dict indicating special execution requirements, such as hardware platforms. |
| `exec_group` | [string](/versions/9.1.0/rules/lib/core/string); default is `'test'`  The name of the exec group that is used to execute the test. |
