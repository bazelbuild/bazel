## toolchain_type

```
toolchain_type config_common.toolchain_type(name, *, mandatory=True)
```

Declare a rule's dependency on a toolchain type.

### Parameters

| Parameter | Description |
| --- | --- |
| `name` | [string](/versions/9.1.0/rules/lib/core/string); or [Label](/versions/9.1.0/rules/lib/builtins/Label); required  The toolchain type that is required. |
| `mandatory` | [bool](/versions/9.1.0/rules/lib/core/bool); default is `True`  Whether the toolchain type is mandatory or optional. |
