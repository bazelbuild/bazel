Provides access to data about a specific toolchain type.
*Note: This API is experimental and may change at any time. It is disabled by default, but may be enabled with `--experimental_platforms_api`*

## Members

* [type_label](#type_label)

## type_label

```
Label ToolchainTypeInfo.type_label
```

The label uniquely identifying this toolchain type.
