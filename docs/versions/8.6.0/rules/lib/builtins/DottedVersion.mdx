title: 'DottedVersion'
---

A value representing a version with multiple components, separated by periods, such as 1.2.3.4.

## Members

* [compare_to](#compare_to)

## compare_to

```
int DottedVersion.compare_to(other)
```

Compares based on most significant (first) not-matching version component. So, for example, 1.2.3 &lt; 1.2.4

### Parameters

| Parameter | Description |
| --- | --- |
| `other` | [DottedVersion](/versions/8.6.0/rules/lib/builtins/DottedVersion); required  The other dotted version. |
